# dynsys-mathos

mathOS — DynSys Engine

This repository contains the mathOS DynSys Engine: a combined formal+runtime framework for dynamical systems research.
Contents: mathOS runtime, Conley 1D/2D grid-cover pipelines, Poincaré map extraction, Lean→mathOS manifest adapter, reports and examples.
Quickstart: Install wheel in dist/ or run examples in examples/. See docs/index.md for full instructions.
